# Portfolio Website: NishSpace

This is a portfolio for Anish Tatke website built using Next.js and Tailwind CSS.

## Features

- Responsive design
- Modern UI/UX
- Fast performance
- Easy to customize

## Technologies Used

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/portfolio-v2.git
    ```
2. Navigate to the project directory:
    ```bash
    cd portfolio-v2
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```

### Running the Development Server

```bash
npm run dev
```
or
```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

```bash
npm run build
```
or
```bash
yarn build
```

## Deployment

You can deploy the project using platforms like Vercel, Netlify, or any other hosting service that supports Next.js.
